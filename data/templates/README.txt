Files in this directory are templates to be used with BDVal.  Copy and
rename the bdval-*" files to match your desired name and place the new files
in the same directory as "bdval.xml" (Do NOT rename bdval.xml).

See http://icb.med.cornell.edu/wiki/index.php?title=BDVAL for information
about the Biomarker Discovery and Validation (BDVal) project.
